## [Lesson5] Prefix Sums  
###  PassingCars : Count the number of passing cars on the road.  
---
Java Solution
#### Task Score : 90   
Correctness : 100 	Performance : 80  
```
class Solution {
    public int solution(int[] A) {
        int zero = 0;
        int count = 0;
        for(int i = 0; i < A.length; i++){
            // A 배열에서 0 카운트해서 0일 아닐 경우 결과값+1
            if(A[i] == 0){
                zero++;
            }else{
                count += zero;
            }
        }
        // 결과값이 1000000000 초과할 경우 -1 로 변경
        if(count > 1000000000){
            count = -1;    
        }
        return count;
    }
}
```

function solution($A) {
